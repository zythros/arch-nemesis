┌───────────────────────────────────────────────────────────────────┐
│  WINDOW MANAGEMENT                                                │
└───────────────────────────────────────────────────────────────────┘
  MOD + j/k             Focus window down/up in stack
  MOD + Shift + j/k     Move window down/up in stack
  MOD + h/l             Decrease/increase master width
  MOD + Shift + h/l     Decrease/increase client height
  MOD + Shift + o       Reset client height
  MOD + q               Kill focused window
  MOD + f               Toggle fullscreen
  MOD + Shift + Space   Toggle floating
  MOD + Tab             View previous tag
  MOD + 0               View all tags
  MOD + Shift + 0       Tag window to all tags
  MOD + 1-9             Switch to tag 1-9
  MOD + Shift + 1-9     Move window to tag 1-9
  MOD + Ctrl + 1-9      Toggle view tag
  MOD + Left/Right      Shift view to prev/next tag
  MOD + Shift + Left/Right  Move window to prev/next monitor
  MOD + , / .           Focus prev/next monitor
  MOD + i               Increase master count
  MOD + n               Decrease master count

┌───────────────────────────────────────────────────────────────────┐
│  LAYOUTS                                                          │
└───────────────────────────────────────────────────────────────────┘
  MOD + Ctrl + F1       Tile []=
  MOD + Ctrl + F2       Spiral [@]
  MOD + Ctrl + F3       Bstack TTT
  MOD + Ctrl + F4       Bstack Horiz ===
  MOD + Ctrl + F5       Grid HHH
  MOD + Ctrl + F6       Centered Master |M|
  MOD + Ctrl + p/m      Cycle layout prev/next

┌───────────────────────────────────────────────────────────────────┐
│  GAPS & BORDERS                                                   │
└───────────────────────────────────────────────────────────────────┘
  MOD + Ctrl + t        Toggle gaps
  MOD + Ctrl + i/d      Increase/decrease all gaps
  MOD + Ctrl + Shift + d  Reset gaps to default
  MOD + Shift + i       Increase inner gaps
  MOD + Ctrl + Shift + i  Decrease inner gaps
  MOD + Ctrl + o        Increase outer gaps
  MOD + Ctrl + Shift + o  Decrease outer gaps
  MOD + Shift + minus   Decrease border width
  MOD + Shift + p       Increase border width

┌───────────────────────────────────────────────────────────────────┐
│  BAR & TABS                                                       │
└───────────────────────────────────────────────────────────────────┘
  MOD + Shift + b       Toggle bar
  MOD + Ctrl + w        Toggle tab mode

┌───────────────────────────────────────────────────────────────────┐
│  SESSION                                                          │
└───────────────────────────────────────────────────────────────────┘
  MOD + Shift + r       Restart chadwm
  MOD + Ctrl + Shift + Return  Zoom (swap with master)

┌───────────────────────────────────────────────────────────────────┐
│  SCREENSHOTS (chadwm)                                             │
└───────────────────────────────────────────────────────────────────┘
  MOD + u               Screenshot selection (maim)
  MOD + Ctrl + u        Screenshot fullscreen (maim)
